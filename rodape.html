Criando Um Rodapé Responsivo

Aqui irei adicionar tags e propriedades que resultarão em um Rodapé Profissional e Responsivo.

Estrutura de um Rodapé HTML


Elemento <footer>

O elemento <footer> é usado para definir o rodapé de uma página ou seção. Ele normalmente contém informações sobre o autor, links de navegação, informações de contato, entre outros. 


    <footer>
        <!-- Conteúdo do rodapé vai aqui -->
    <footer>

Contêiner Principal

Dentro do <footer> utilizamos um contêiner principal para organizar as diferentes seções do rodapé. O elemento <div class="footer-container"> serve como um contêiner flexível para agrupar e estilizar as seções internas. 


    <div class="footer-container">>
        <!-- Seções do rodapé vão aqui -->
    </div>

Seção "Sobre Mim"

Esta seção fornece informações sobre o autor do site. É comum incluir um título e um parágrafo descrevendo brevemente o autor ou o propósito do site. 


    <div class="footer-section">
        <h3>Sobre Mim</h3>
        <p>Adicione um texto sobre você<p>
    </div>
  • <div class="footer-section"> Cria um contêiner para a seção.
  • <h3> Define o título da seção.
  • <p> Contém um parágrafo com a descrição.

Seção "Links Úteis"

Esta seção inclui uma lista de links que podem ser úteis para os visitantes do site, como links para outras páginas do site. 


    <div class="footer-section">
        <h3>Links Úteis</h3>
        <ul>Adicione um texto sobre você<ul>
            <li><a href="#">Início</a></li>
            <li><a href="#">Serviços</a></li>
            <li><a href="#">Contato</a></li>
            <li><a href="#">Sobre</a></li>
        </ul>
    </div>
  • <ul> Cria uma lista não ordenada.
  • <li> Cria um item de lista.
  • <a href="#"> Cria um link(a) direcionado(href) a uma página específica localizada em(#) alguma pasta.
  • <li><a href="#">.....</a></li> Cria um item de lista com um link(<a>).

Seção "Contato"

Esta seção oferece informações de contato, como email, telefone e endereço. 


    <div class="footer-section">
        <h3>Contato</h3>
        <p>Email: Adicione seu email.<p>
        <p>Telefone: Adicione seu número<p>
        <p>Endereço: Adicione seu endereço<p>
    </div>
  • <p> Cada parágrafo contém uma linha de informação de contato.

Seção "Siga-me"

Nesta seção, incluímos links para redes sociais com ícones. Cada ícone é uma imagem que funciona como um link para a respectiva rede social. 


    <div class="footer-section social-media">
        <h3>Siga-me</h3>
        <a href="#"><img src="Whatsapp" alt="Whatsapp"></a>
        <a href="#"><img src="X" alt="X"></a>                        
        <a href="#"><img src="Instagram" alt="Instagram"></a>                        
        <a href="#"><img src="LinkedIn" alt="LinkedIn"></a>
    </div>
  • <img src="..."> Adiciona uma imagem(img) localizada em tal pasta(src).
  • <alt> (texto-alternativo) Se a imagem não puder ser carregada por algum motivo, o texto alternativo será exibido no lugar da imagem, dando ao usuário uma ideia do que deveria estar ali..
  • <a href="#"><img src="..." alt="..."></a> Cada parágrafo contém uma imagem(icone) com link, que ao ser clicado será direcionado para o contato em cada rede social.

Seção Inferior do Rodapé

A seção inferior do rodapé normalmente contém informações de direitos autorais ou outros avisos legais. 


    <div class="footer-bottom"">
        <© Todos os direitos reservados>Contato</h3>
    </div>
  • <div class="footer-bottom"> Cria um contêiner para a seção inferior.
  • <p> Contém um parágrafo com o texto de direitos autorais.

Estilizando o Rodapé Com CSS

Agora vamos estilizar a estrutura que criamos no html, afinal de contas o que da "vida" a página é o CSS.

Body


    body {
        font-family: Arial, sans-serif;
        margin: 0;
        padding: 0;
        background-color: #cdcbcb;
        line-height: 1.6;
    }
  • font-family:Arial, sans-serif; Define a família de fontes para Arial, com uma alternativa genérica sans-serif
  • margin: 0; Remove a margem padrão do corpo
  • padding: 0; Remove o preenchimento padrão do corpo
  • background-color: #cdcbcb; Define a cor de fundo da página
  • line-height: 1.6; Define a altura da linha para melhorar a legibilidade do texto

Estilo para o Rodapé


    footer {
        background-color: #333;
        color:white;
        padding: 20px 0;
        position: relative;
        width: 100%;
    }
  • background-color: #333; Define a cor de fundo do rodapé como cinza escuro */
  • color: white; Define a cor do texto como branca
  • padding: 20px 0; Adiciona 20px de preenchimento no topo e na base
  • position: relative; Define o posicionamento relativo para o rodapé
  • width: 100%; Define a largura do rodapé como 100% da largura do contêiner pai

Contêiner Principal do Rodapé


    .footer-container {
        display: flex;
        flex-wrap: wrap;
        justify-content: space-between;
        max-width: 1200px;
        margin: auto;
        padding: 0 20px;
    }
  • display: flex; Define o contêiner como um flexbox
  • flex-wrap: wrap; Permite que os itens flexbox sejam distribuídos em várias linhas
  • justify-content: space-between; Distribui o espaço restante uniformemente entre os itens
  • max-width: 1200px; Define a largura máxima do contêiner
  • margin:auto Centraliza o contêiner horizontalmente
  • padding: 0 20px; Adiciona 20px de preenchimento à esquerda e à direita

Seção Individual do Rodapé


    .footer-section {
        flex: 1;
        min-width: 250px;
        margin: 10px;
    }
  • flex:1 Permite que cada seção cresça proporcionalmente para ocupar o espaço disponível.
  • min-width: 2500px; Define a largura mínima para cada seção.
  • margin: 10px; Adiciona uma margem de 10px ao redor de cada seção.

Título das Seções do Rodapé


    .footer-section h3 {
        margin-bottom: 15px;
    }
  • margin-bottom: 15px; Adiciona uma margem inferior de 15px para separar o título do conteúdo.

Parágrafos e Listas nas Seções do Rodapé


    .footer-section p, .footer-section ul {
        margin: 0;
        padding: 0;
    }
  • margin: 0; Remove a margem padrâo.
  • padding: 0; Remove o preenchimento padrâo.

Listas nas Seções do Rodapé


    .footer-section ul {
        list-style: none;
    }
  • list-style: none; Remove os marcadores de lista.

Itens das Listas nas Seções do Rodapé


    .footer-section p, .footer-section ul {
        margim-bottom: 10px;
    }
  • margin-bottom: 10px; Adiciona uma margem inferior de 10px para separar os itens da lista.

Links nas Listas das Seções do Rodapé


    .footer-section ul li a {
        color: white;
        text-decoration: none;
        transition: color 0.5s;
    }
  • color: white; Define a cor do texto como branca.
  • text-decoration: none; Remove a sublinha dos links.
  • transition: color 0.5s; Adiciona uma transição suave de 0.5 segundos para mudança de cor.

Efeito Hover nos Links


    .footer-section ul li a:hover {
        color: red;
    }
  • color: red; Muda a cor do texto para vermelho quando o mouse passa sobre o link.

Ícones de Redes Sociais


    .social-media a {
        display: inline-block;
        margin-right: 10px;
        margin-bottom: 10px;
    }
            
    .social-media img {
        width: 24px;
        height: 24px;
    }
  • display: inline-block Exibe os links como blocos inline.
  • margin-right: 10px; Adiciona uma margem direita de 10px para separar os ícones horizontalmente.
  • margin-left: 10px; Adiciona uma margem inferior de 10px para separar os ícones verticalmente.
  • width: 24px; Define a largura dos ícones como 24px.
  • height: 24; Define a altura dos ícones como 24px.

Seção Inferior do Rodapé


    .footer-bottom {
        text-align: center;
        padding: 10px 20px;
        background-color: #222;
        margin-top: 20px;
    }
  • text-align: center; Centraliza o texto.
  • padding: 10px 20px; Adiciona 10px de preenchimento no topo e na base, e 20px à esquerda e à direita.
  • background-color: #222; Define a cor de fundo como um cinza ainda mais escuro.
  • margin-top: 20px; Adiciona uma margem superior de 20px para separar do resto do conteúdo.

Estilo Responsivo


    @media (max-width:768px) {
        .footer-container {
            flex-direction: column;
            align-items: center;
        }
            
        .footer-section {
            text-align: center;
            margin: 20px 0;
        }
    }
  • flex-direction: column; Altera a direção dos itens flex para coluna em telas menores.
  • align-items: center; Centraliza os itens em telas menores.
  • text-align: center; Centraliza o texto nas seções em telas menores.
  • margin: 20px 0; Adiciona uma margem vertical de 20px para espaçamento.

Explicação Adicional do CSS

Estilo Global do body

  • Define a fonte padrão, remove margens e preenchimentos padrão, e define um fundo cinza claro com uma altura de linha confortável.

    • Estilo para o <footer>

  • Define o fundo escuro, texto branco, preenchimento para espaçamento interno, e garante que o rodapé ocupe toda a largura da página.

    • Contêiner Principal .footer-container

  • Utiliza flexbox para organizar as seções, permitindo que elas se ajustem e se distribuam uniformemente. Centraliza o contêiner na página.

    • Seções Individuais .footer-section

  • Cada seção é configurada para crescer proporcionalmente e tem uma largura mínima para garantir legibilidade.

    • Títulos das Seções .footer-section h3

  • Adiciona espaçamento inferior para separar visualmente o título do conteúdo.

    • Parágrafos e Listas

  • Remove margens e preenchimentos padrão para garantir consistência no layout.

    • Listas .footer-section ul

  • Remove os marcadores padrão das listas.

    • Itens das Listas .footer-section ul li

  • Adiciona espaçamento entre os itens da lista.

    • Links das Listas .footer-section ul li a

  • Estiliza os links para serem brancos, sem sublinha, e com uma transição suave para a cor vermelha ao passar o mouse.

    • Ícones de Redes Sociais .social-media a e .social-media img

  • Define o tamanho e o espaçamento dos ícones para redes sociais.

    • Seção Inferior .footer-bottom

  • Centraliza o texto, adiciona preenchimento para espaçamento interno, e define um fundo ainda mais escuro para contraste.

    • Estilo Responsivo

  • Altera a direção do flexbox para coluna e centraliza o texto em telas menores, garantindo que o rodapé permaneça legível e bem organizado em dispositivos móveis.
